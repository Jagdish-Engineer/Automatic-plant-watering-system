# Soil Moisture Monitoring System

## Overview
An IoT-based system that detects soil moisture levels and alerts when moisture drops below a threshold.

## Features
- Real-time moisture detection
- Alert system for low moisture
- Efficient water usage

## Components Used
- Soil moisture sensor
- Microcontroller
- Buzzer

## Applications
- Smart irrigation
- Home gardening
